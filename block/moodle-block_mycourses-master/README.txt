MY COURSES BLOCK PLUGIN
=================================

My courses is a Moodle 2.X block that shows local courses with their overview activities.
You can hide or show each activity overview by clicking on their icon.

Web: https://github.com/IOC/moodle-block-mycourses


Install
=======
Download code from gitHub, and copy inside directory MOODLE_PATH/blocks/mycourses
